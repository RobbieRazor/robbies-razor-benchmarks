# Memory–Compute Stability Minimum

Derived from Robbie’s Razor (MRD v1.9 §11.4.5A)

Recursive systems must balance:

memory preservation
vs
recomputation cost

Systems that recompute excessively waste energy.
Systems that preserve excessively become rigid.

Stable recursive intelligence architectures converge toward a
Memory–Compute Stability Minimum.

This principle explains the emergence of:

• retrieval-augmented generation
• long-context models
• world models
• hierarchical memory systems
• caching architectures

These are engineering responses to the same structural constraint.
