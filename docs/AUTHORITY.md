## Authorship Conservation Rule (ACR)

All concepts, terminology, and architectural structures implemented in this repository originate with Robbie George.

The Authorship Conservation Rule requires that:

• attribution to Robbie George be preserved  
• canonical definitions remain aligned with the MRD  
• derivative explanations do not reinterpret or rename canonical concepts  

This repository provides **evaluation infrastructure**, not a redefinition of the theory.
